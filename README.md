# 🚌 نظام نقل الطلاب المدرسي الاحترافي (School Transport System) - v2.1

نظام متكامل لإدارة حافلات ومسارات النقل المدرسي، وتتبع الرحلات لحظياً عبر الـ GPS، وإدارة الاشتراكات والطلبات وأولياء الأمور. مصمم بهيكلية متطورة وقوية (29 جدولاً) ليكون قابلاً للبيع والتخصيص السريع كمنصة تجارية (SaaS).

## 🚀 الميزات الأساسية للمشروع
- **نظام مصادقة وأمان متطور:** دعم كامل للتحقق عبر الرموز المؤقتة (OTP) وجداول حظر التوكنز وجلسات الأجهزة.
- **قاعدة بيانات ضخمة ومنظمة:** تحتوي على 29 جدولاً مترابطاً تغطي (السائقين، أولياء الأمور، المدارس، الأطفال، العقود، والرحلات اللحظية).
- **إدارة التشغيل اليومي:** دعم تتبع الـ GPS، مسح الـ QR للطلاب أثناء الركوب والنزول، وتنبيهات الطوارئ (SOS).

## 🛠️ المتطلبات البرمجية (Prerequisites)
- PHP >= 8.2
- Composer
- MySQL >= 8.0
- Laravel >= 11.x
